# selflook - 张益达的个人主页

我的个人网站，部署在 GitHub Pages。

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- 响应式设计，适配桌面与移动端
- 滚动动画、毛玻璃导航等现代 UI 效果

## 本地预览

```bash
# 用任意静态服务器即可，例如：
npx serve .
# 或
python3 -m http.server 8000
```

然后打开 http://localhost:8000

## 部署

推送到 GitHub 仓库的 `main` 分支，在 Settings > Pages 中启用 GitHub Pages 即可自动部署。
